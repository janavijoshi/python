GIT configuration setup
1. git config --global user.name "<your-username>"
2. git config --global user.name

3. git config --global user.email "<your-email>"
4. git config --global user.email

5. git init: initialises a repository (folder)
6. git status: displays the current status of files and commits in the git repo
7. git add .: adds all the changes to the git repo

8. git commit -m "<commit message>": provide a message to your commit
9. git branch -M main: changes root branch to "main"
10. git branch: displays all the branches and highlights the current branch

11. git push -u origin <branch-name>
12. git merge <branch-name>:merging the branch with main, only works locally will not show in the remote repo
